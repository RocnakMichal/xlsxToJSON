# xlsxToJSON
## Aplikace pro převod .xlsx souborů na soubory .JSON

Uživatel zadá jména souborů, která chce převést. Nemusí udávat příponu .xlsx tu si případně program dodá sám. Pokud chce převést více souborů najednou musí je oddělit čárkou. V případě, že má požadovaný soubor ve svém názvu čárku je nutné jej napsat do "". Do uvozovek je též nutné napsat názvy souborů, které se jmenují jako klíčová slova. 
### Klíčovími slovy jsou: 
  - vse-Převede všechny soubory, které jsou ve stejném adresáři jako program
  - napoveda-Uživatel získá nápovědu k programu
  - kratky vypis-Hodnoty v buňkách, které se rovnají `null` se nevypíší
  - dlouhy vypis-Vypisou se i hodnoty, kde je null (defaultně nastaveno)
  - konec-Ukončí program
